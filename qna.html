Question-1: What are the different ways to select an element in the DOM?

Answer: There are 5 ways to select element in the DOM. They are:

  1. getElementById
  2. getElementsByClassName
  3. getElementsByTagName
  4. querySelectorAll
  5. querySelector

    6. Here we can navigate only one element by getElementById and querySelector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:
innerHTML: This property can read any HTML code with Tags in the DOM element.
innerText: This property can return any HTML text without CSS hidden styles.
textContent:This property can return any HTML text with visible or hidden CSS styles.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a mechanism in Javascript where an event listener is connected to a parent element to handle event for all of its child element.

Question-4: What is event bubbling in the DOM?

Answer: Event bubble is a process in the DOM where an event on a targeted element then step by step bubbles up through its parent element in the DOM tree untill it arrives the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: We can create an element in Javascript by using document.createElement() method. then we can add the created element with the parent element using parent.appendChild('new created element'). if we want to remove any child element from parent element, we can use parent.removeChild('targeted child element')
